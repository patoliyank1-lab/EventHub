# 📌 EventHub – Event Management Dashboard

## 🎯 Final Deliverable

A **beautiful, responsive Event Management Dashboard** that demonstrates:

* ✅ Strong HTML structure
* ✅ Creative use of CSS styling
* ✅ Utility-based design with Bootstrap
* ✅ Structured, interactive React components
* ✅ Responsive design for mobile and desktop

---

## 📋 Requirements

### ✅ Event Management

#### 📋 Event Table

Display a table of **upcoming events** with the following columns:

* Event Name
* Date & Time
* Location
* Status (Active / Inactive)
* Actions (Edit | Delete)

---

### ➕ Add New Event Form

#### Fields:

* **Event Name** *(required)*
* **Date & Time** *(with date-time picker)*
* **Location** *(required)*
* **Status** *(dropdown: Active / Inactive)*

#### Behavior:

* On submission, dynamically add the event to the event list table

---

### ✏️ Edit Event Inline

* Allow inline editing of event fields directly within the table row

---

### ❌ Delete Event

* Add a delete button with a confirmation prompt
* Remove the event from the list upon confirmation

---

## ✅ Attendee Management (Bonus)

* Provide an option to **view attendees** for each event
* Display attendees in a **Bootstrap modal popup**
* Use mock attendee data:

  * Name
  * Email
