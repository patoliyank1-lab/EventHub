📌 Project Name: EventHub – Event Management Dashboard
🎯 Final Deliverable
A beautiful, responsive Event Management Dashboard that showcases:
 ✅ Strong HTML structure
 ✅ Creative use of CSS styling
 ✅ Utility-based design with Bootstrap
 ✅ Structured, interactive React components
 ✅ Responsive design for mobile and desktop
📋 Requirements
✅ Event Management
📋 Display a table of upcoming events with the following columns:
Event Name
Date & Time
Location
Status (Active / Inactive)
Actions (Edit | Delete)
➕ Add New Event Form:
Fields:
Event Name (required)
Date & Time (with date-time picker)
Location (required)
Status (dropdown: Active / Inactive)
On submission, dynamically add event to the event list table.
✏️ Edit Event Inline:
Allow inline editing of event fields directly within the table row.
❌ Delete Event:
Add a delete button with a confirmation prompt to remove the event from the list.
✅ Attendee Management (Bonus)
For each event, display an option to view attendees in a Bootstrap modal popup.
Mock attendee data (Name, Email) is used for demonstration.
✅ Responsive UI
Use Bootstrap grid system for layout.
Include a navigation bar:
Home | Events | Add Event | Profile | Logout
Fully responsive for mobile, tablet, and desktop.
✅ Real Interactivity
State management using React Hooks: useState, useEffect.
Bootstrap modals, tables, forms, buttons, and alerts.
Show success/error alerts after add/edit/delete actions.
Form validation (required fields, date must be in the future).
⏱ Time Estimation
6–8 hours (includes building functionality, form validation, responsive design polish, and handling edge cases)
